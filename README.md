# Line Notify
## use
you have to run 
```
npm install
```
to install the node modules
then run
```
npm start
```
to run the code, before run the code you have to write the LINE_TOKEN in the .env file below
```
EXPRESS_PORT = 
LINE_TOKEN = 
```
finally you can go to *http://*yourip** to enter the message
